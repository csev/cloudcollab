Welcome to CloudCollab.com. This is a login server for IMS Simple Learning Tools Interoperability.

This server uses IMS Simple LTI to launch you into:

  • CloudSocial-enabled web content with the "pop-up" LMS.
  • The CloudCollab Demo LMS and tool set which uses IMS Simple LTI as its Single-Sign-On
The steps are: (1) log in to this server using your Google credentials and (2) go to the launch page and launch content.

When you log in to CloudCollab you will be asked for your Google ID/PW. Look at the URL - this is really Google. Rest assured that the only thing that CloudCollab knows about you after you log in is your E-Mail address and nickname and the fact that you have logged in. We are given no other data from Google.

This site also is participating in the Beta of Google Friend Connect. Please join the FriendConnect site at left to get the best experience as some of the CloudCollab tools are Friend Connect Widgets.

This site is written in Python and is running on the Google Application Engine.
