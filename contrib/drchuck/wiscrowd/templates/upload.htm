IMS Common Cartridge / SimpleLTI Test Harness

This form allows you to upload a IMS Common Cartridge with a SimpleLTI descriptor. The test harness will read through the cartridge searching for SimpleLTI descriptors. For each descriptor it will make a small form to allow you to launch the content contained in the descriptor. If the test harness encounters errors, it will let you know what it things is wrong with the cartridge.

This only tests SimpleLTI entries in the cartridge for validity - it assumes the rest of the cartridge already validates.
