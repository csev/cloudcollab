Welcome to the very simple test framework for Simple IMS Learning Tools Interoperability (LTI). This looks like an LMS - but really it is intended to be launched from another LMS using the IMS SimpleLTI protocol. The IMS SimpleLTI protocol is a REST-style protcol allowing an LMS system and a Learning Tool to establish trust, exchange identity information, user roles, and exchange information about course context. Once this information is exchanged - tools have access to the information and can operate accordingly.

If you log in to this system, it looks really complex - because it is not just a log in - on the "login" screen - you effextively completely set all of the security, user, course, and role information and then launch into the software. This is not fake - the form does a POST to the IMS Simple LTI framework built into this application - the *exact same POST* as comes from a normal LMS such as Sakai when Sakai is doing a Simple LTI launch.

You can change any of the values on the log in form to test the system witgh different users, roles, courses, etc. If you don't change anything - you log in as Chuck and a student in SI301 at the University of Michigan.

The IMS Learning Tools Interoperability and Simple LTI are being developed by the www.imsglobal.org. If you are interested in standards and interoperability for teaching and learning technology you may want to become involved in IMS.
