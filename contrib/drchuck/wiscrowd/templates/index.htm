Welcome to the demo of the CloudCollab framework for building applications using Simple IMS Learning Tools Interoperability (LTI).

This looks like an LMS - but really it is intended to be a collection of tools launched from another LMS using the IMS SimpleLTI protocol.

If you log in to this system, it looks really complex - because it is not just a log in - on the "login" screen - you effectively completely set all of the security, user, course, and role information and then sign all of that information and then launch into the software. The form does the actual REST-style POST to the IMS Simple LTI framework built into this application. This is the *exact same format* as a POST that comes from a normal LMS such as Sakai when Sakai is doing a Simple LTI launch.

You can change any of the values on the log in form to test the system with different users, roles, courses, etc. If you don't change anything - you log in as Chuck and a student in SI301 at the University of Michigan.

The IMS Learning Tools Interoperability and Simple LTI are being developed by the www.imsglobal.org. If you are interested in standards and interoperability for teaching and learning technology you should become involved in IMS.
