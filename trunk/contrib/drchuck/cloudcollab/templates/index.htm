Welcome to CloudCollab.com. This is a login server for IMS Simple Learning Tools Interoperability. This server can launch you into CloudSocial-enabled content or into tools that can be launched using IMS Simple Learning Tools Interoperability.

This site also is participating in the Beta of Google Friend Connect. Please join the site at left to get the best experience as some of the CloudCollab tools are Friend Connect Widgets.

CloudCollab.com uses Google as its authentication - so when you log in to CloudCollab you will be asked for your Google ID/PW. Look at the URL - this is really Google. Rest assured that the only thing that CloudCollab knows about you after you log in is your E-Mail address and nickname and the fact that you have logged in. We are given no other data from Google.

This site is written in Python and is running on the Google Application Engine.
